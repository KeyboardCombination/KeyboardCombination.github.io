Hashbrown

Easy to use GUI downloader for Roblox assets. Made in Python and QT. I started this project after my friends couldn't figure out how to download Roblox assets from the command line. Shoutouts to Modnark for the code for downloading Roblox assets.
Note: This only works with uncopylocked/onsale assets.

 Try it out!
